CYF Project: Build an image carousel

Details: Image carousel: Allow the user to cycle through two set of images In this project we have two sets (each set contains five pictures) of pictures which are animals and landscapes. We have options to show either animals or landscapes pictures in cycle.
In CYF Blocks:
  1. In static HTML, we have an image tag and four buttons for Animals, Landscapes, Next and Previous. The id’s of all tags are same with their names.
  2. We have 7 blocks (in CYF Blocks) including click on; run, Animals, Landscapes, Next, and Previous and two functions.
  3. The two functions were named Animals_fun and Lanscapes_fun. In each of this function we have two variables named count (type: integer) and image_item (type: string).
  4. In each function, we have five if conditions. The value of count will be between 0 to 4. For each value of the count, we set the value of image_item as an address of one of the pictures.
  5. At start (when run is clicked).
    1. We set count as zero.
    2. Calling Animals_fun.
    3. We use find element with id image.
    4. Set the attribute src as image_item.
  6. When the element with id Animals is clicked
    1. Find the element with id Animals.
    2. Set the attribute backgound to black colour.
    3. Set the attribute color to white color
    4. Find the element with id Landscapes.
    5. Set the attribute backgound to white colour.
    6. Set the attribute color to black color.
    7. Set the variable choose to 1.
  7. When the element with id Landscapes is clicked.
    1. Find the element with id Landscapes.
    2. Set the attribute backgound to black colour.
    3. Set the attribute color to white color
    4. Find the element with id Animals.
    5. Set the attribute backgound to white colour.
    6. Set the attribute color to black color.
    7. Set the variable choose to 0.
  8. When the element with id Next is clicked
    1. If count ≤ 4 do change count by -1
    2. If count < 0 set change count by 4
    3. If choose = 1 do Animal_fun.
    4. If choose = 0 do Landscape_func
    5. Find element with id image
    6. Set the attribute src to image_item.
When the element with id Prevous is clicked If count $\le$ 3 do change count by 1 If count $ge$ 4 set change count by 0 If choose $=$ 1 do Animal_fun. If choose = 0 do Landscape_func Find element with id image And set the attribute src to image_item.
